bta512_business_analytics_with_programming
==========================================

Portland State University course workspace for BTA 512:
Business Analytics with Programming.

Current contents:
  bta512 - cardiointellect - analytics project.ipynb
    Main project notebook for the CardioIntellect analytics project.

  bta512 - cardiointellect - analytics project.pdf
    Exported PDF version of the notebook/report.

What this project appears to be:
  A predictive analytics course project focused on heart disease prediction,
  comparing machine-learning approaches against a defined business/decision use case.

Why it matters in a portfolio:
  This is the most directly health-oriented of the three BTA projects. It is
  useful for medical, digital health, or healthcare-app contexts where
  predictive support models and interpretable classification comparisons matter.

Primary artifact:
  The notebook is the source of truth for analysis logic and narrative.

Suggested use:
  Open the notebook if you want to review or extend the analysis.
  Use the PDF for quick reading or submission-style review.
